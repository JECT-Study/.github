# 젝트는 대한민국의 IT 동아리입니다
우리는 IT 기술과 문제해결적 사고로 더 좋은 미래를 만들어내기 위해 활동하는 동아리입니다.</br>
동아리원들을 기수제로 연 2회 모집하고, 디지털 서비스나 제품을 기획-개발-디자인하는 팀 협업 프로젝트를 중점적으로 진행하고 있어요.</br>
그 밖에도 전문분야에 대한 기술 학습과 네트워킹을 병행하며 함께 성장하는 것을 목표로 하고 있습니다.</br>
모든 젝트 구성원들은 협업과 성장의 즐거움을 널리 알리며 IT 생태계의 선순환을 위해 활동합니다.</br>
